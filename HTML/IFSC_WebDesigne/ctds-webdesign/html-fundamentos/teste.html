ICliente

Uma Interface é um contrato que define um conjunto de métodos que uma classe deve implementar. As interfaces são usadas para definir comportamentos que podem ser compartilhados por classes diferentes, sem forçar uma relação de herança entre elas. Elas permitem a criação de código mais flexível e desacoplado.

Note: Leia atentamente a documentação desta classes para desfrutar dos recursos oferecidos

Interface ICliente

A interface ICliente define dois métodos:

Métodos

String getDados(): Este método retorna uma string que representa os dados do cliente.
String getDados(String observacao): Este método retorna uma string que representa os dados do cliente, possivelmente incluindo alguma observação específica passada como argumento.
